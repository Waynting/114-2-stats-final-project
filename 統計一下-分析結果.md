## 一、研究概要

### 1.1 研究目標

本研究承襲企劃書所提出的 10 個研究問題（RQ1 ～ RQ10），探討 YouTube Shorts 的標題、標籤、影片長度、發布時間與頻道規模對觀看數的影響。在此之上，本研究進一步擴充企劃書未提的兩項分析：

1. **內容貼標體系**（Layer 1 ～ Layer 4）：以 YouTube 演算法視角（`topic_categories`）、規則式關鍵字字典、jieba TF-IDF、LDA 主題模型四層方法，把「標題關鍵字」與「標籤類型」這兩個語意性 RQ 變成可操作變數。
2. **跨類別比較**：透過 ANOVA、OLS（含類別 dummy 與交互作用）、Logistic、卡方檢定，比較九個使用者人工分類在控制訂閱規模後的觀看溢價，並驗證企劃書研究背景「教育類因高留存獲長尾推撥」的假設。

### 1.2 資料規模

| 維度 | 數量 |
|---|---:|
| 使用者人工類別 | 9 |
| 頻道 | 28 |
| 影片總數（含長片） | 約 5,800 |
| 分析樣本 Shorts（duration_sec ≤ 180） | **4,798** |

九個類別分別為：餐廳/食品品牌推廣（`restaurant_brand`）、介紹食物（`food_review`）、運動（`sports`）、科普（`pansci`）、遊戲（`games`）、時事（`news`）、籃球幹片（`basketball_meme`）、籃球網紅（`basketball_influencer`）、Kpop（`kpop`）。`restaurant_brand` 與其餘八類別在 Phase 3 階段以同一輪 channel-compare 抓取時間戳 `20260527-072439` 重新抓取，確保跨類別比較使用同期資料。

### 1.3 分析方法

| 方法 | 對應 RQ | 主要用途 |
|---|---|---|
| 描述統計 + 相關分析 | RQ1, 3, 6 | 標題長度／標籤數量／影片長度與觀看的單變量關係 |
| ANOVA / Kruskal-Wallis | RQ1, 3, 9 | 類別內頻道差異、跨類別差異 |
| Mann-Whitney U | RQ6 | Shorts vs 長片、頻道兩兩比較 |
| 卡方檢定 | RQ2 | has_*（內容字典命中）與 top-quartile 觀看的關聯 |
| Logistic 迴歸 | RQ2 | has_問句、has_搞笑迷因等對「進入 top quartile」的勝算比 |
| OLS 迴歸 | RQ1, 3, 6, 7, 8, 10 | 控制變數下標題、長度、tag、訂閱、發片頻率對 log(view) 的線性關係 |
| TF-IDF + Cosine | RQ4, 5 | 頻道間內容相似度、類別內聚分數 |
| LDA 主題模型 | RQ4 | 跨類別主題潛在分群、語料層次驗證使用者分類 |

統計檢定一律採取雙尾，顯著水準 $\alpha = 0.05$。OLS 模型結果列出係數、$p$ 值、$R^2$、VIF。

### 2.2 內容貼標方法（企劃書未提出，本研究擴充）

企劃書 RQ2、RQ4、RQ5 涉及「標題關鍵字」「標籤類型」「標題-標籤一致性」三項語意性問題，原則上是定性概念。為將其量化為可檢定變數，本研究建立四層貼標體系：

**Layer 1：YouTube `topic_categories` 解析**
- 由 API 直接回傳的 Wikipedia URL 取尾段（如 `https://en.wikipedia.org/wiki/Food` → `Food`），代表 YouTube 演算法視角對每部影片的主題標。每影片可有多 topic。
- 用途：對照使用者人工分類，檢驗 RQ4「標籤類型」的演算法視角實況。

**Layer 2：規則式關鍵字字典 v1.0**
- 10 個 bucket（`has_教學教育`、`has_挑戰對決`、`has_搞笑迷因`、`has_開箱評測`、`has_互動誘導`、`has_情緒詞`、`has_品牌推廣`、`has_食物`、`has_運動`、`has_問句`），每 bucket 含 5 ～ 30 個關鍵字。
- 文字來源：`title + tags` 串接後 lowercase。
- 命中規則：bucket 中任一關鍵字出現於文字 → 該 bucket = 1。
- 用途：對應 RQ2「標題關鍵字」的可檢定版本。

**Layer 3：jieba + TF-IDF 頻道級指紋**
- Tokenizer：jieba v0.42.1（中文）。
- 每頻道一份 document（concat 所有 Shorts 標題），跨頻道計算 IDF，每頻道取 Top 15 keyword。
- 用途：頻道層級內容指紋；跨頻道 cosine 相似度。

**Layer 4：LDA 主題模型**
- 全體 Shorts 標題為語料（4,789 篇），TF-IDF（`max_features=3000`, `min_df=5`, `max_df=0.6`）作為輸入。
- $K \in \{6, 8, 10, 12\}$ 中以 perplexity 選最佳。最佳 $K = 6$，perplexity = 1231.41。
- 用途：跨類別主題潛在分群；驗證使用者分類在語料上的同質性。

### 2.3 控制變數

跨類別 OLS 主要控制以下變數：

- `log_sub = log10(subscriber_count + 1)` — 訂閱規模（RQ7）
- `title_length` — 標題字元數（總長度，含中英日）
- `tag_count` — `tags` 欄以 `|` 分隔計算（缺值記為 0）
- `duration_sec` — 影片長度（RQ6）
- `videos_per_month` — 頻道歷史平均發片頻率 = `channels.video_count / months_since_created`（RQ10）

依變數採取 `log_view = log10(view_count + 1)`，因 `view_count` 跨類別量級差異 70 倍以上、嚴重右偏（多數類別偏度 $> 2$、峰度 $> 13$），對數轉換是跨類別比較的必要前置。

## 三、各類別內部分析摘要

### 3.1 餐廳／食品品牌推廣（restaurant_brand）

樣本：3 個頻道（Ju茱莉亞林、Lisa Sung、蜡笔小锋）、323 部 Shorts。訂閱中位數 66,400（九類最小之一），觀看中位數 237,886。`view_per_subscriber` 中位數 **3.6 為九類最高**，超越 basketball_influencer 的 2.3。

跨類別 OLS（M2）中該類別 dummy coef = **+0.640**（$p < .001$）對應觀看為 baseline 的 **4.36 倍**，僅次於 games（+0.856 / 7.18 倍）。Reg/02 早期分析顯示頻道間 ANOVA $F = 119.6$、$p = 1.6 \times 10^{-39}$ — Ju茱莉亞林平均 1.88M 觀看，Lisa Sung / 蜡笔小锋約 0.34M，差距達 5 倍。Layer 1 主題以 Lifestyle (0.76) / Food (0.45) 為主，Food 命中率明顯低於 food_review 的 0.91，顯示三品牌頻道偏「生活風格 + 探訪」格式而非純美食評論。Layer 2 命中 `has_食物 = 0.170`、`has_品牌推廣 = 0.006`，雙標籤特性確認。內聚 gap +0.034 倒數第二，類別邊界鬆散；LDA 主導 T4（牛排／挑戰／回顧／往期），與 food_review 主導 T5 分屬不同 cluster — **演算法視角下「餐廳品牌」與「美食評論」並非同一主題**。

限制：3 頻道 323 部樣本仍小；類別內主題異質性大；發片頻率分布廣（1.4 ～ 40.9 部/月）使 RQ10 在類別內變動極大。

### 3.2 介紹食物（food_review）

樣本：7 個頻道、965 部 Shorts，類別內樣本量第二大。觀看 mean = 514,373、median = 120,935。

頻道間 ANOVA $F = 9.31$、$p = 6.07 \times 10^{-10}$ — 顯著但 F 值是三類別中最低（內部相對均質）。觀看排序：智明 Jimmypsd（mean 1.0M）> うまぐるめ（978K）> This is Ken（513K）> 吃貨豪豪（357K）> 冬冬（71K）> 貓跪妃（52K）> E ating（40K）。標題長度 ANOVA $F = 7.11$, $p = 8.64 \times 10^{-4}$ — **中等長度（mean 684K）最佳**，呈倒 U 型，與 sports 的單調正向不同。

Layer 2 命中最具預測力：智明 Jimmypsd `has_問句` 命中 93.9%（「（地名）一天要花多少」公式），該頻道平均觀看為類別冠軍。卡方檢定 `has_問句` $\chi^2 = 176.44$, $p = 2.91 \times 10^{-40}$，Logistic 顯著項 `has_問句` $\beta = +2.12$、勝算比 **8.3 倍**進入 top quartile。Layer 3 顯示各頻道內容指紋互異（台中夜市、東京日本、屋台拉麵、限量在地、連鎖速食、費用挑戰、隱藏推薦各自獨立）。

OLS：M1 $R^2 = 0.030$、加入 channel dummy 後 $R^2 = 0.066$（$\Delta R^2 = +0.036$，中等頻道效應）。`title_length` 在 M1、M2 皆顯著正向（$\beta \approx 255K$, $p = 0.002$）。整體解釋力低反映「該影片拍到哪個主題」的隨機性主導。

限制：日文頻道うまぐるめ的 `has_食物` 因字典只有中文 token 僅命中 1.5%，造成假性低；類別 RQ3 因 962/965 無 tag 完全不可分析；觀看分配 std/mean ≈ 3.4 極偏。

### 3.3 運動（sports）

樣本：4 個頻道（NBA、MLB、挖掘肌讲健身、老赵闹健身）、1,172 部 Shorts — **九類別樣本量最大**。觀看 mean = 74,532、median = 20,401，跨類別倒數第二（僅高於 basketball_meme）。

頻道間 ANOVA $F = 26.42$、$p = 1.56 \times 10^{-16}$。觀看排序反直覺：**老赵闹健身（mean 117K）> NBA（94K）> MLB（72K）> 挖掘肌讲健身（15K）** — 中文健身內容平均觀看高於 NBA／MLB 官方頻道（兩者訂閱 7M+ ／ 24M+）。

標題長度 ANOVA $F = 24.58$, $p = 3.5 \times 10^{-11}$ — **長標題平均觀看為短標題的 3.4 倍**（短 34K → 長 117K），呈強單調正向。Layer 1 完美對應：NBA Basketball 96% / Sport 97%，MLB Baseball 96%，兩個健身頻道 Physical_fitness 86 ～ 91%。Layer 2 `has_運動` 卡方 $\chi^2 = 32.71$, $p = 1.07 \times 10^{-8}$ — 但實質為頻道差異代理（MLB 99.3% vs NBA 47.1%）。

OLS：M1 $R^2 = 0.055$、M2 = 0.065（$\Delta R^2 = +0.010$，頻道效應較弱）。`title_length` 顯著正向 $\beta = +30K$，$duration_sec$ 顯著負向 $\beta = -16K$（越短觀看略多）。

限制：英文 NBA/MLB 標題以單詞分隔、中文健身標題以字元密集，`title_length` 變數混雜兩種語言計算單位；殘差偏態極大（Skew = 6.03）；MLB 7.2M 訂閱 vs 老赵闹健身 76.5K 但觀看反低，訂閱規模非主導因子。

### 3.4 科普（pansci，單頻道）

樣本：@PanScitw 一個頻道、154 部 Shorts、46 部長片。觀看 mean = 184,527、median = 124,584。

由於單頻道無法做頻道間 ANOVA，本類別改以 Shorts vs 長片兩樣本 Mann-Whitney U：`view_count` $p = 0.0977$ ns、`like_count` $p = 0.0132$ * — **Shorts 在按讚數上顯著優於長片**，觀看數雖較高但邊界 ns。

時間分段 ANOVA（依 published_at 切三等分）$F = 4.51$, $p = 0.0126$ * — 中期 Shorts 表現最佳（mean 246K）、近期最差（136K）；可能反映演算法或內容週期效應。

Layer 1：YouTube 系統將 PanSci 主要標為 **Health 22.5% / Society 19.5% / Knowledge 5.5%**，**「Education」topic 0 命中**。YouTube 視 PanSci 為「健康與社會議題」而非「正規教育」。Layer 2：`has_問句` 命中率 35.7%（長片 65.2%）— PanSci 的核心 hook 是「以問句製造好奇」（「為什麼…？」「什麼決定了…？」）；`has_教學教育` 僅 1.3%，反映字典 v1.0 未涵蓋問句型教學動詞。

OLS（M1）：`view_count ~ duration_sec + title_length + tag_count`，$R^2 = 0.006$、$F$ test $p = 0.808$ — **三個結構性變數完全不能預測 PanSci Shorts 的觀看數**。`view_count` 變異主要由話題本身、首頁熱度、外部曝光等未觀測因素解釋。

限制：單頻道無 channel dummy 設計；發布時段 9 成集中 UTC 11:00（台灣 19:00），「發布時段」此變數失效；長片 $n = 46$ 偏小；Lablodol 缺值問題此類別不涉及。

### 3.5 遊戲（games）

樣本：2 個頻道（camman18、GothamChess）、308 部 Shorts。觀看 mean = **5,179,427 為九類最高**（遠超第二的 547K）— 兩頻道皆為國際大型英文頻道（10M+ 訂閱），與其他類別不在同量級。

兩頻道 Mann-Whitney U（camman18 vs GothamChess）：`view_count` $p = 5.5 \times 10^{-29}$、`title_length` $p = 5.1 \times 10^{-40}$、`tag_count` $p = 1.0 \times 10^{-68}$ — 所有變數極顯著差異。camman18 訂閱 10.8M / 平均觀看 6.67M；GothamChess 7.58M / 2.49M。

Layer 1 完美對應：camman18 Video_game_culture 100% / Role-playing_video_game 93.4% / Action-adventure_game 84.3%（YouTube 把 Minecraft 一律歸為 RPG / Action-adventure）；GothamChess Strategy_video_game 6.4%（Entertainment 20.9% / Hobby 17.3% 主導），西洋棋的演算法視角為「策略娛樂」。Layer 2 `has_教學教育` 與 `has_挑戰對決` 在 camman18 均 100% 命中（實為 tag 重複導致的偽命中，不是內容真的全是教學）。

OLS：M1 $R^2 = 0.208$、M2 = 0.208（加 channel dummy 後幾乎無提升，但設計矩陣 condition number = $5.7 \times 10^{15}$ — `channel × tag_count` 嚴重共線，無法分離）。

限制：兩頻道規模都極大，跨類別比較必須正規化（這也是本研究選用 log_view 與 view_per_subscriber 三重正規化的主因）；tag 重複度極高，tag_count 變異主要來自「有 vs 沒有」而非「多 vs 少」。

### 3.6 時事（news）

樣本：5 個頻道（Johnny Harris、Vox、志祺七七、喵耳電波、Real Fake）、964 部 Shorts。觀看 mean = 533,436、median = 149,132。

頻道間 ANOVA $F = 92.91$, $p = 8.42 \times 10^{-67}$ — **跨類別觀看差異最劇烈**：Johnny Harris mean 1.73M ／ Vox 42K ／ Real Fake 961（差距 1,800 倍）。

標題長度 ANOVA $F = 6.15$, $p = 2.21 \times 10^{-3}$ — 中等最佳（489K → 717K → 390K），呈倒 U 型。標籤數量 ANOVA $F = 97.13$, $p = 3.80 \times 10^{-39}$ — **無 tag 反而觀看最高**（1.71M vs 4+ 的 637K vs 1-3 的 139K），主要因 Johnny Harris 等英文長片頻道短片化通常不打 tag。

Layer 1 嚴重錯位：news 類在 YouTube 視角下 Politics 命中率僅 0.19、Entertainment 反高達 0.39。Layer 2 `has_問句` $\chi^2 = 17.52$, $p = 2.84 \times 10^{-5}$，Logistic 顯著項 `has_問句` $\beta = -0.72$（$p < .001$）— **在 news 類問句反而與較低觀看相關**（因 top quartile 由 Johnny Harris 英文長片碎片化主導，其 has_問句僅 19.8%）。`has_搞笑迷因` $\beta = -0.37$（$p = 0.036$）類似方向。

OLS：M1 $R^2 = 0.053$、M2 = 0.283（$\Delta R^2 = +0.230$，**三類別中最大頻道效應**）。`title_length` 在 M2 顯著負向（$\beta = -115K$, $p = 0.038$）。

限制：Real Fake n=8 嚴重不足；類內語言（英文 vs 中文）異質性極大；`has_搞笑迷因` 在喵耳電波 100% 命中是字典 spurious（頻道描述含「迷因」字）。

### 3.7 籃球幹片（basketball_meme）

樣本：2 個頻道（俊鸿TV、卷毛懂个球）、304 部 Shorts。觀看 mean = 76,862、median = 21,015。

Mann-Whitney U：俊鸿TV vs 卷毛懂个球，`view_count` $p = 2.12 \times 10^{-19}$ ***，俊鸿TV 顯著高（120K vs 26K）。俊鸿TV 訂閱 69,300（卷毛 5,030 的 14 倍），但觀看中位數差 4 倍 — 訂閱可解釋大部分但非全部差距。

Layer 1：俊鸿TV Basketball 90.2% / Sport 86.6% — YouTube 認定「硬核籃球」；卷毛懂个球 Basketball 67.9% / Humour 30.0% / Entertainment 27.9% — 「娛樂為主的籃球段子」。Layer 2：俊鸿TV `has_運動` 96.3% / `has_搞笑迷因` 65.2%；卷毛懂个球 `has_搞笑迷因` 67.9% / `has_運動` 5.0% — 兩頻道在「搞笑迷因」命中率接近但「運動」極不同。

卡方檢定（top_quartile）：`has_運動` $\chi^2 = 21.04$, $p = 4.5 \times 10^{-6}$；`has_挑戰對決` $p = 0.039$；`has_搞笑迷因` $p = 0.050$。

OLS：M1 $R^2 = 0.221$、M2 = 0.286。`title_length` 在 M1 顯著正向（$\beta = +58K$, $p < .001$），加 channel dummy 後失效（$p = 0.21$）— **標題長度其實是頻道風格的代理**。`tag_count` 在 M1 顯著負向（$\beta = -27.6K$, $p < .01$）— 與直覺相反。

限制：兩頻道訂閱差 14 倍、文字風格極不同；俊鸿TV 與曾舔舔🕊️ 有合拍系列，類別邊界模糊。

### 3.8 籃球網紅（basketball_influencer）

樣本：2 個頻道（曾舔舔🕊️、肖他）、310 部 Shorts。觀看 mean = 105,868、median = 34,480。訂閱中位數 21,800（九類最小），`view_per_subscriber` 中位數 2.3（第二高）— **「小頻道高觸及」典範類別**（在 restaurant_brand 加入前是冠軍）。

兩頻道 Mann-Whitney U：曾舔舔🕊️ vs 肖他，`view_count` $p = 2.0 \times 10^{-15}$ ***。曾舔舔🕊️ mean 140K、肖他 46K；訂閱 21,800 vs 9,770（2.2 倍）但觀看差 3 倍。

Layer 1：肖他 Basketball 84.7% / Sport 96.4% — 「純粹體育」；曾舔舔🕊️ Basketball 73.9% / Lifestyle 27.6% / Humour 14.1% — 「娛樂化籃球」。Layer 2：曾舔舔🕊️ `has_運動` 78.9% / `has_教學教育` 28.6%；肖他 `has_運動` 57.7% / `has_問句` 21.6%。

卡方檢定 `has_運動` $\chi^2 = 11.84$, $p = 5.8 \times 10^{-4}$ 顯著預測 top quartile。OLS：M1 $R^2 = 0.037$、M2 = 0.063；`title_length` 在 M1 顯著（$\beta = +35.7K$, $p = 0.001$）、加頻道 dummy 後失效 — 同 basketball_meme「頻道風格驅動」。

限制：`tags` 欄全 NaN，所有 tag-related 分析失效；曾舔舔🕊️ 用繁體 + 台灣 KOL 體系，肖他用簡體 + 中國校園情境，文化脈絡差異未控制。

### 3.9 Kpop（kpop）

樣本：2 個頻道（K-潮流、kukuo 쿠쿠）、298 部 Shorts。觀看 mean = 145,826、median = 50,548。

Mann-Whitney U：K-潮流 vs kukuo，`view_count` $p = 0.044$ * — **kukuo 訂閱僅 K-潮流 7%（5,290 vs 76,600）但平均觀看 2.3 倍**（233K vs 103K）— Kpop 類別最反直覺發現，訂閱數與 Shorts 觀看脫鉤。

Layer 1：K-潮流 Entertainment 69% / Lifestyle 13.5%；kukuo Entertainment 81.6%。兩頻道 **Music topic 命中皆 < 4%** — K-Pop 反而被 YouTube 歸到泛娛樂類，「音樂類」標籤幾乎不出現。Layer 2：K-潮流 `has_教學教育` 7.0% / `has_搞笑迷因` 4.0%；kukuo `has_問句` 9.2% / `has_食物` 5.1% / `has_情緒詞` 5.1` — Kpop 字典整體命中率極低（< 10%），字典 v1.0 對 K-Pop 偶像文本覆蓋不足。

卡方檢定：所有 bucket $p > 0.05$ — **內容字典標籤對 Kpop 高觀看無預測力**。OLS：M1 $R^2 = 0.029$、M2 = 0.043；連續變數全 ns；C(channel)[kukuo] +136.5K *（$p = 0.035$）。

跨頻道 cosine 內聚 gap +0.185 **九類別最高** — 兩 K-Pop 頻道用詞極相似（韓國／偶像／演唱會等專有名詞），但 LDA 仍把 K-Pop 主導歸到 T1（食物 / 日本料理混合 topic）— 顯示 LDA 在中英韓混雜語料上的 topic 邊界更受語言別影響。

限制：Lablodol 缺值（無 @handle）；v1.0 字典未涵蓋「偶像/應援/直拍/MV」buckets；kukuo $n = 98$ 較小，mean 易被少數爆款拉高。

## 四、跨類別整合分析

### 4.1 跨類別 ANOVA：三種正規化下「類別」的區分力

對全體 4,798 部 Shorts 跑「類別」單因子 ANOVA / Kruskal-Wallis，三種觀看度量結果如下：

| 度量 | F | Kruskal H | 結論 |
|---|---:|---:|---|
| view_count (raw) | 385.3 | 1,780.4 | $p \approx 0$，類別極顯著，但被訂閱規模污染 |
| log_view = log10(view+1) | **406.4** | 1,780.4 | $p \approx 0$，F 略升、分佈更穩定 |
| view_per_subscriber | 30.9 | **1,963.0** | $p = 1.2 \times 10^{-47}$，F 大降但 Kruskal 仍極高 |

**結論**：
- 不論何種正規化，類別皆顯著影響 Shorts 表現。
- 從 raw 到 vps，F 下降約 13 倍 — **約 92% 的「raw 類別差異」可由訂閱規模解釋**，剩下 8% 是內容/類別固有差異。
- $\log_{10}$ 轉換下類別 F 最大，**`log_view` 為最適合的跨類別比較指標**。
- vps 的 Kruskal > ANOVA F — 各類 vps 分佈嚴重非常態，中位數比較較可靠。

### 4.2 大 OLS 模型：M1（無 category）vs M2（含 category dummy）

**M1**：$\log\text{view} \sim \log\text{sub} + \text{title\_length} + \text{tag\_count} + \text{duration\_sec}$，$n = 4{,}798$。

| 變數 | coef | $p$ |
|---|---:|---:|
| Intercept | +3.346 | $< .001$ |
| $\log\text{sub}$ | **+0.331** | $< .001$ |
| title_length | −0.0048 | $< .001$ |
| tag_count | +0.0065 | $< .001$ |
| duration_sec | −0.0010 | $< .001$ |

- **$R^2 = 0.200$**，Adj $R^2 = 0.199$；VIF 全 $\approx 1$（無共線性）。
- 訂閱每多 1 個量級（10×），$\log\text{view}$ 平均增加 0.33 → 觀看數增為原 $10^{0.33} \approx 2.14$ 倍。

**M2**：M1 加入 `C(category)`，**$R^2 = 0.494$**，Adj $R^2 = 0.493$。**$\Delta R^2 = +0.294$** — 控制訂閱規模後，類別 dummy 仍貢獻 29.4% 的觀看變異。

類別效應（baseline = `basketball_influencer`）：

| 類別 | coef | $p$ | $10^{coef}$（倍數） |
|---|---:|---:|---:|
| games | **+0.856** | $< .001$ | 7.18 |
| restaurant_brand | **+0.640** | $< .001$ | 4.36 |
| food_review | +0.147 | 0.001 | 1.40 |
| pansci | +0.022 | 0.766 | $\approx$ baseline |
| kpop | −0.145 | 0.006 | 0.72 |
| news | −0.234 | $< .001$ | 0.58 |
| basketball_meme | −0.445 | $< .001$ | 0.36 |
| sports | **−0.909** | $< .001$ | 0.12 |

**核心解讀**：
- 在同樣訂閱規模、同樣標題長度、同樣 tag 數、同樣時長之下，**games Shorts 觀看為 basketball_influencer 的 7.18 倍；restaurant_brand 為 4.36 倍；sports 僅 0.12 倍**。
- $\log\text{sub}$ 在 M2 仍保留 +0.314 的強顯著主效應 — 訂閱與類別兩者各占獨立貢獻。
- pansci 控制訂閱後與 baseline 等價，**直接挑戰「教育類獲長尾推撥」假設**（詳見 §6.1）。
- `title_length` 從 M1 的 −0.0048 翻轉為 M2 的 +0.0034（正向、顯著）— **Simpson 悖論的典型例子**：跨類別異質性導致 M1 的長標題效應方向被反向（某些長標題類別觀看本身就高，未控制類別時被誤判）。

### 4.3 類別 × tag_count 交互作用（M3）

模型：$\log\text{view} \sim \log\text{sub} + \text{duration\_sec} + C(\text{category}) \times \text{tag\_count}$，$R^2 = 0.495$。

顯著交互項：

| 交互項 | coef | $p$ | 解讀 |
|---|---:|---:|---|
| food_review × tag_count | **−0.243** | 0.045 | 食物類加 tag 反而降觀看（tags 全 NaN 頻道把線拉成負相關） |
| games × tag_count | +0.050 | 0.010 | 遊戲類每多 1 tag, log_view +0.050（× 1.12） |
| news × tag_count | +0.044 | 0.024 | 新聞類加 tag 微正向 |
| restaurant_brand × tag_count | +0.019 | 0.339 ns | 餐廳品牌類 tag 效應微弱不顯著 |

**結論**：標籤數量的觀看效應**並非普世均勻**，僅在 games / news 有正向，food_review 反而負向（受 tags 全 NaN 頻道影響極大）。「多打 tag = 多觀看」這條建議只在特定類別成立。

### 4.4 YouTube 自動 topic 與使用者人工分類的錯位

每影片可有多 topic，計算每類別中該 topic 至少命中一次的比率：

| 使用者類別 | YouTube 主要 topic（命中率） | 對應強度 |
|---|---|---|
| games | Video_game_culture 0.85 / Role-playing_video_game 0.60 | **完美對應** |
| food_review | Food 0.91 / Lifestyle 0.98 | **完美對應** |
| restaurant_brand | Lifestyle 0.76 / Food 0.45 | 對應，Food 命中較低 |
| sports | Physical_fitness 0.58 / Sport 0.36 / Lifestyle 0.59 | 部分對應，被健身頻道稀釋 |
| basketball_meme | Basketball 0.80 / Sport 0.65 / Humour 0.16 | 對應 |
| basketball_influencer | Basketball 0.78 / Sport 0.77 / Humour 0.09 | 對應 |
| news | Politics 0.19 / Society 0.29 / Entertainment 0.39 | 中等對應、新聞主題非 YouTube 標準 topic |
| pansci | Health 0.27 / Society 0.25 / Knowledge 0.07 | **顯著錯位**，0% Education |
| kpop | Entertainment 0.73 / Music 0.04 | 一致但泛化，0% Music topic |

**結論**：
1. 使用者人工類別與 YouTube 演算法視角**部分一致、部分嚴重錯位**。最一致 games（細分到 Video_game_culture）、其次 food_review（Food / Lifestyle 命中 0.9+）。
2. **pansci 錯位最劇**：YouTube 從未把 PanSci 標為 `Education`，主要落 Health / Society / Knowledge — 對「教育類獲長尾推撥」假設構成挑戰。
3. **news Politics 命中率僅 0.19**：對「新聞」這個使用者類別，YouTube 反而最常標為 Entertainment（0.39）— 演算法視角下新聞與娛樂的距離比想像中近。

### 4.5 跨類別內容標籤效應（has_* × category）

對 `has_<bucket> × C(category)` 跑 Logistic：$P(\text{top\_quartile\_view}) \sim \text{has\_bucket} \times C(\text{category})$。部分類別命中率 0/1 導致 singular matrix，退回 additive 模型：

| Bucket | OR | $p$ | 結論 |
|---|---:|---:|---|
| **has_問句** | **1.52** | $1.8 \times 10^{-5}$ | **跨類別最普世**：問句型標題使進入 top quartile 機率提升約 52% |
| has_搞笑迷因 | 1.19 | 0.19 | 無跨類別一致效應 |
| has_情緒詞 | $\approx$ 1 | $> 0.27$ | 跨類別大致一致無顯著 |
| has_教學教育 | — | — | news × has_教學教育 顯著負向（coef = −2.07, $p = 0.019$）— 新聞類用「教學」反而降觀看 |
| has_挑戰對決 | — | — | 因樣本集中無法檢出 |

**結論**：跨類別最普世的標題策略是「問句型 hook」，其餘標籤策略普遍**類別依賴**。

## 六、對企劃書研究背景假設的回應

### 6.1 「教育類因高留存獲長尾推撥」假設

**部分否定**：

1. **M2 大模型結果**：pansci dummy coef = +0.022（$p = 0.766$ ns）— 在控制訂閱規模後，pansci 觀看數與 baseline（basketball_influencer）幾乎相同，**並無額外的「教育類加成」**。
2. **YouTube 演算法視角錯位**：pansci 沒有任何一部影片被自動歸到 `Education`，主要落在 Health（22.5%）/ Society（19.5%）/ Knowledge（5.5%）— **YouTube 本身就不把 PanSci 視為「教育類」**，所謂「教育類獲長尾推撥」的前提是否成立有疑。
3. **LDA 視角下無獨立教育 topic**：pansci 在 K=6 LDA 中主導 T0（英文/中文常用詞混雜 topic），與 games / news / sports 共享 — pansci 的「教育類」標籤是使用者主觀。

**支持的部分**：

- pansci `has_問句` 命中 35.7%（Shorts）、65.2%（長片）；跨類別 logit additive `has_問句` OR = 1.52 *** — **問句型 hook 確實是高觀看 Shorts 的共通特徵**，PanSci 的策略與此一致。
- pansci `view_per_subscriber` 中位數 0.1 雖低，但 mean view 在九類中排名第五（185K），考量單頻道 1.19M 訂閱規模，仍屬中規中矩 — 大型專業頻道在 Shorts 平台確有長尾優勢的微弱訊號，但效應遠不如「品牌效應 + 訂閱規模」。

**研究啟示**：「教育類獲長尾推撥」的論述若成立，前提須先定義「YouTube 系統內所認知的教育類」而非「使用者主觀認知的科普頻道」。在資料層次上，PanSci 並未享有跨類別額外溢價；它與 baseline 等價的結果，意味教育類在 Shorts 平台上的演算法待遇與一般頻道並無顯著差異。

## 七、最反直覺發現（報告口頭簡報的「主論點」候選）

1. **多發片 $\neq$ 多觀看**：videos_per_month 與 $\log\text{view}$ 顯著負相關（coef = −0.0013, $p < .001$ 影片層級）。MLB、NBA 把長片切片量產，每支 Shorts 觀看反而被稀釋；Lisa Sung 1.4 部/月、智明 0.79 部/月才能在每支累積觀看。

2. **title_length 的 Simpson 悖論**：M1（無類別）顯示標題越長觀看越少（coef −0.0048），M2（含類別 dummy）翻轉為正向（+0.0034）— **是否「長標題降觀看」完全取決於是否有控制類別**。

3. **訂閱規模在 kpop 與觀看反向**：kukuo 訂閱僅 K-潮流 7% 但平均觀看 2.3 倍；Kpop Shorts 的推送高度依賴演算法外推爆款，訂閱不是主因子。

4. **`has_問句` 是少數普世有效的標題策略**：跨類別 OR = 1.52 ***，是 9 類別共享的高觀看特徵；其他 has_* bucket 均為類別依賴。

5. **PanSci 的觀看變異 99.4% 與標題包裝無關**：OLS（duration + title_length + tag_count）$R^2 = 0.006$、$F$ test $p = 0.808$ — 三個結構性變數完全無預測力，觀看主要由話題本身、演算法推送等未觀測因素決定。

6. **restaurant_brand 是「最高觸及效率」類別**：view/sub 中位數 3.6 為九類最高，超越 basketball_influencer 的 2.3；在 M2 中類別溢價 coef = +0.640（× 4.36 倍）僅次於 games — 在控制訂閱規模後，「餐廳/品牌推廣」這類內容仍享有獨立的觀看溢價。

7. **YouTube 演算法把 PanSci 視為 Health/Society 而非 Education**：教育類 topic 0% 命中，新聞類 Politics 命中率僅 19% — **演算法視角下「教育類」「新聞類」這些概念可能不存在**，使用者直覺與系統視角嚴重錯位。

## 八、限制與未來方向

### 8.1 方法論限制

1. **規則字典 v1.0 對日文/英文 token 不公**：日文頻道うまぐるめ的 `has_食物` 命中僅 1.5%（漢字「食」未在日文標題出現），形成假性低；Kpop 偶像名（韓文/英文音譯）未在字典內。
2. **`title_length` 跨語言計算偏誤**：英文以單詞分隔、中文以字元密集；NBA/MLB title_length 50 約 8 ～ 10 英文詞，中國健身頻道 50 約 25 ～ 30 漢字 — 跨類別字元數不可直接比較。
3. **小樣本頻道不穩定**：Real Fake $n = 8$ 在 logistic 與 ANOVA 中只能保留為觀察值；其極端值會干擾 news 內聚分數估計。
4. **jieba 詞表未針對科普/籃球專業詞訓練**：PanSci 的「量子／突觸」、籃球的「進攻效率」等專有詞被切碎，LDA 因此把 pansci 歸到 T0「英文/中文常用詞」topic。
5. **Logistic 跨類別交互模型多次 singular**：因部分類別在某 bucket 上命中為 0/1，互動矩陣秩不足，退回 additive 估計，無法解析「該 bucket 在哪個類別最有效」的細節。
6. **發片頻率測量單一**：videos_per_month 用 `channels.video_count / months_since_created` 是「歷史平均」，不反映最近 3 個月發片密度的變化；對轉做 Shorts 的頻道會嚴重低估近期密度。
7. **LDA 主導因子是語言而非主題**：K=6 結果 4 個英文/中文常用詞頻道共享 T0（含 games / news / sports / pansci），顯示 LDA 在中英混雜語料上的 topic 邊界更受語言別與格式詞影響，主題代表力被稀釋。
8. **未控制季節性效應**：資料時段為 2026-05 抓取的近 200 部上傳，未控制節日、政治事件、演算法政策變動等時間因素。
9. **未驗證演算法推播權重**：本研究僅觀察觀看結果，未取得 YouTube Studio 的曝光點擊率（CTR）、平均觀看時長（AVD）等內部指標 — 「教育類獲長尾推撥」應在演算法層而非觀看結果層直接驗證。

### 8.2 樣本缺口

| 缺口 | 影響 |
|---|---|
| Lablodol（Kpop）handle 未補 | Kpop 類別僅 2 頻道，無法達計劃書原列 3 頻道目標 |
| Real Fake $n = 8$（news） | 描述統計保留、檢定受極端值干擾 |
| basketball_influencer tags 全 NaN | 該類別所有 tag-related 分析失效 |
| food_review 962/965 無 tag | 類別內 RQ3 無實際分析價值 |
| restaurant_brand 僅 3 頻道 323 部 | 類別內聚分數 +0.034 偏低，類別內主題異質性高 |

### 8.3 未來方向

1. **擴充 restaurant_brand 頻道**：目前 3 個頻道，若擴充至 6 ～ 8 個品牌頻道，可更穩健估計「商業推廣 vs 一般類別」差異。
2. **加入嵌入式語義模型**：用多語預訓練 sentence embedding（如 multilingual-MiniLM）取代 TF-IDF cosine，可消除中英文割裂問題、提高 LDA 復現使用者分類的能力。
3. **時間序列分析**：把 published_at 切月，看每類別 Shorts 觀看隨時間的演變、是否有「節日效應」或「演算法政策變動」的痕跡。
4. **頻道效應分離**：在 M2 之上加入 `C(channel_id)` 隨機效應（mixed-effects model），把「品牌效應」與「內容效應」徹底分離。
5. **更細的內容 bucket**：擴充 `content_keywords.yaml` 加入「時事」「政治」「街拍」「韓流」「偶像」等專有 bucket，並對日文/韓文 token 各自訓練（或 OpenCC 統一中日漢字）。
6. **取得 YouTube Studio 內部指標**：CTR、AVD、impression source 等才能直接驗證演算法權重假設。

## 結語

本研究透過 4,798 部 Shorts、9 個使用者類別、28 個頻道的大樣本資料，並以四層內容貼標體系把企劃書中的語意性 RQ 轉化為可檢定變數，最終得到以下核心結論：

1. **類別在控制訂閱規模後仍顯著貢獻觀看變異**（$\Delta R^2 = +0.294$）；games 與 restaurant_brand 享有最大類別溢價，sports 反而受切片量產拖累。
2. **問句型標題是少數跨類別普世有效的策略**（OR = 1.52 ***），其餘標題關鍵字、標籤策略均為類別依賴。
3. **「教育類獲長尾推撥」假設在資料層次未獲支持**：pansci 控制訂閱後與 baseline 等價，且 YouTube 演算法從未將 PanSci 歸為 Education topic。
4. **發片頻率與觀看數呈負相關**，顛覆「多發片演算法多推」直覺。
5. **訂閱規模在 kpop 等類別與觀看脫鉤**，YouTube Shorts 的演算法推送邏輯與一般長片頻道顯著不同。

這些結論為短影音創作者提供了具體的優化方向：採用問句型 hook、精修少發、選擇高溢價類別、控制訂閱規模對自身內容形態的適配性，比盲目跟風熱門標籤策略更具決定性。
