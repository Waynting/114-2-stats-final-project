## 簡報核心敘事弧

> 整份簡報的故事線：**「人工分類的觀看差異有多大？ → 是訂閱規模造成的嗎？ → 控制訂閱後仍有 29.4% 變異來自類別 → 那是什麼類別特性？文字策略？標籤？發片頻率？ → 揭露三個反直覺發現 → 推翻企劃書原假設（教育類長尾）→ 結論與限制」**

把每頁串成一個「揭謎」流程，而不是「條列分析項目」。

## P4 — 分析方法概覽

**講者要點**
- 依變數：`log_view = log10(view_count + 1)`（解決右偏，跨類別差 67 倍）
- 控制變數：`log_sub`、`title_length`、`tag_count`、`duration_sec`、`videos_per_month`
- 八類方法：描述統計、ANOVA / Kruskal–Wallis、卡方、Logistic、OLS、TF-IDF cosine、LDA、Jaccard 相似度
- 四層內容貼標：
  - **Layer 1** YouTube `topic_categories`（API 直回，平台視角）
  - **Layer 2** 規則字典 v1.0（10 個 bucket：has_問句、has_教學教育、has_搞笑迷因 …）
  - **Layer 3** jieba TF-IDF（頻道語意指紋、類別內聚分數）
  - **Layer 4** LDA K=6（潛在主題分群，K 由 perplexity 選取）

**視覺需求**
- 四層金字塔圖（由淺入深：平台 → 規則 → 統計 → 潛在）
- 旁邊一個迷你流程圖：原始資料 → 控制變數 → 四層特徵 → 八類方法 → 結論

**講者備註**
- 強調「四層體系」是本研究的方法論貢獻：把「標題策略」這種模糊概念，逐層量化為可進回歸模型的變數

## P5 — 發現 1：類別溢價（控制訂閱後仍有 29.4% 變異）

**講者要點**
- M1（不含類別）：R² = 0.200
- M2（含類別 dummy）：R² = 0.494，**ΔR² = +0.294**
- 訂閱效應：`log_sub` coef = +0.314（訂閱 ×10 → 觀看 ×2.06）
- 類別溢價排序（M2 dummy coef）：
  - games **+0.856**（×7.18）
  - restaurant_brand **+0.640**（×4.36）
  - food_review +0.147（×1.40）
  - pansci +0.022（ns，baseline）
  - sports **−0.909**（×0.12）
- 解讀：控制訂閱後仍有 29.4% 觀看變異來自「**類別屬性本身**」——這是後續所有討論的起點

**視覺需求**
- 主圖：M2 類別 dummy 係數柱狀圖（依係數排序，正向藍色 / 負向紅色，星號顯著性標註）
- 副圖：M1 R²=0.200 vs M2 R²=0.494 對比甜甜圈圖

**講者備註**
- 預告：sports 為什麼是 −0.909？不是內容差，是高頻量產稀釋（P8 會回來解釋）
- 預告：pansci coef ≈ 0 是後面推翻「教育類長尾假設」的關鍵證據（P10）

## P6 — 發現 2：標題策略的類別依賴性 + Simpson 悖論

**講者要點**
- **跨類別最普世策略**：`has_問句` OR = 1.52（***，p = 1.8×10⁻⁵）
- **類別特異性極強**：
  - food_review `has_問句` OR = **8.3**（智明 Jimmypsd 命中 93.9%）
  - news `has_教學教育` β = **−2.07**（顯著負向，p = 0.019）
  - kpop、basketball_meme 等：規則字典字面命中極低
- **Simpson 悖論示範**：title_length
  - M1（單變量）coef = **−0.0048**（負向）
  - M2（加 category 控制後）coef = **+0.0034**（正向翻轉）
  - 解讀：「長標題拉低觀看」是錯覺——其實是「長標題的類別本身觀看就低」（如英文 news vs 中文 sports 短標題）
- 結論：**標題建議不能跨類別通用**，研究界把「特徵 → 觀看」當普世關係本身就是統計謬誤

**視覺需求**
- 主圖：M1 vs M2 係數對比的對偶長條（title_length 在 M1 紅色負、M2 藍色正，特別標註箭頭）
- 副圖：各類別 `has_問句` 命中率熱力圖（深綠 = 命中率高）

**講者備註**
- 這頁是整份報告的「方法論亮點頁」——讓老師看到我們知道 Simpson 悖論並且用實例展示
- 口語化解釋：「以為長標題不好，是因為長標題的人本來在差類別」

## P7 — 發現 3：標籤策略的類別異質性

**講者要點**
- tag_count 主效應：OLS coef = +0.0065（***）——表面上「多打 tag 多觀看」
- 加入類別交互後反轉：
  - food_review × tag_count：coef = **−0.243**（*，反向）
  - games × tag_count：coef = **+0.050**（*）
- food_review 反向原因：965 部中 962 部 tag = 0（資料稀疏），少數有 tag 的反而是觀看較低的小頻道
- 標題–標籤 Jaccard 一致性：整體 ρ = +0.057（**，弱相關）；僅 games（ρ = +0.390）、kpop、sports 顯著
- 解讀：創作者用 tag 做 SEO、標題做 hook，兩者邏輯不同步——「tag 多 = 觀看多」**不是普世建議**

**視覺需求**
- 主圖：類別 × tag_count 交互效應斜率圖（每條線一個類別，斜率正負一目了然）
- 副圖：標題–標籤 Jaccard ρ 各類別散點 + 顯著性

**講者備註**
- food_review tag 反向是「資料稀疏 + 觀看高的人不打 tag」的混淆效應，而不是「打 tag 真的傷害觀看」
- 口頭強調統計推論 ≠ 因果，這也是後面限制章節要呼應的點

## P8 — 發現 4：發片頻率的負相關（最反直覺）

**講者要點**
- RQ10 核心結果：`videos_per_month` coef = **−0.0013**（影片層級 p<.001***、頻道層級 p=0.022*）
- 高頻量產 → 每支觀看反而低：
  - MLB：1,266 部/月 → 每支觀看極低
  - NBA：317 部/月 → 同上
- 低頻精修 → 每支觀看高：
  - 智明 Jimmypsd：0.79 部/月 → 觀看中位破百萬
  - Lisa Sung：1.4 部/月 → 同樣高位
- 推測機制：高頻量產稀釋演算法推送權重；精修少發累積口碑與完成率
- 結論：**「演算法獎勵勤勞」的直覺被資料否定**

**視覺需求**
- 主圖：videos_per_month vs log_view 散點圖（每個點是一個頻道，按類別上色）
- 副圖：MLB / NBA vs 智明 / Lisa Sung 對比卡片（頭像 + 數字 + 結論）

**講者備註**
- 這頁是「最容易讓觀眾驚訝的一頁」，留 30 秒讓觀眾消化
- 強調：這是觀察性資料，不能直接建議創作者「少發片」，但提供了強烈的觀察證據

## P9 — 發現 5：YouTube 演算法視角 vs 人工分類錯位

**講者要點**
- 完美對應的類別：
  - games：`Video_game_culture` 命中 0.85
  - food_review：`Food` 命中 0.91
- 嚴重錯位的類別：
  - **pansci**：`Education` 命中 **0%**（被歸到 Health 0.27、Society 0.25）
  - **news**：`Politics` 命中 **0.19**（被歸到 Entertainment 0.39）
  - **kpop**：`Music` 命中 **0.04**（被歸到 Entertainment 0.73）
- 解讀：使用者直覺的「教育類」「新聞類」「音樂類」與 YouTube 系統內部的標籤體系**不同步**
- 對推薦演算法的啟示：創作者可能被推送到「不對的觀眾池」而不自知

**視覺需求**
- 主圖：9 類別 × 主要 YouTube topic 的命中率熱力圖（橫軸類別、縱軸 topic、深淺表命中率）
- 副圖：pansci / news / kpop 三個錯位案例的圓餅圖並列

**講者備註**
- 帶到 P10：pansci 在 YouTube 系統內根本不被識別為「Education」，這直接動搖了企劃書「教育類獲長尾推送」的前提

## P10 — 發現 6：推翻「教育類長尾」企劃書假設

**講者要點**
- 企劃書原假設：**教育類因留存高、完整看完率高，獲演算法長尾推送**
- 三層證據逐一檢驗，全部否定：
  - **OLS 層**：M2 中 pansci dummy coef = **+0.022**（p = 0.766 ns）——控制訂閱後**無額外加成**
  - **YouTube topic 層**：pansci `Education` 命中 **0%**——演算法根本不認為它是教育類
  - **LDA 層**：K=6 主題模型中**無獨立教育主題**，pansci 主導 T0（與 games/news 共享的英文常用詞主題）
- 部分支持的細節：`has_問句` 在 pansci 命中 35.7%，跨類別 OR=1.52——pansci 確實使用問句 hook 策略
- 結論：「教育類獲長尾」在本資料層次**未獲支持**

**視覺需求**
- 三欄並列「✗ ✗ ✗」結構：OLS、YouTube topic、LDA 三層證據，每欄一張小圖
- 底部紅色橫條：「企劃書假設 → 證據否定」

**講者備註**
- 這是整份報告**最學術價值高的一頁**——展示「我們不只跑數據，我們檢驗自己原本的信念」
- 老師通常很喜歡「研究結果推翻原假設」的論證

## P13 — LDA 主題模型與類別內聚分數

**講者要點**
- LDA K=6 最佳（perplexity 選取）
- **語言 > 主題**現象：games / news / sports / pansci **共享 T0**（英文+常用詞主題）——LDA 在多語料上主導因子是語言而非主題
- 食物相關分裂為兩主題：food_review T5（評論）vs restaurant_brand T4（品牌探訪）
- 籃球相關分裂為兩主題：basketball_meme T2（幹片）vs basketball_influencer T5（網紅）
- 類別內聚分數（TF-IDF cosine gap）排序：
  - kpop **+0.185**（最高，K-pop 詞彙專屬性高）
  - sports +0.079、food_review +0.064
  - basketball_influencer **+0.025**（最低，混語料）
- 結論：LDA 復現人工分類的程度**有限**——這是方法論限制，也是未來改進方向

**視覺需求**
- 主圖：LDA 主題分佈堆積長條圖（橫軸 9 類別、縱軸主題佔比 0-1、6 種顏色）
- 副圖：類別內聚分數橫向長條圖（由高到低排列）

**講者備註**
- 強調這頁展示「我們知道方法的局限」——對 LDA 結果採取了批判態度，不是盲信
- 預告 P14：方法限制章節會繼續延伸

## P14 — 方法論限制與資料缺口

**講者要點**
- **內容貼標限制**
  - 規則字典 v1.0 對日韓文 token 不公（うまぐるめ `has_食物` 命中僅 1.5%）
  - title_length 跨語言計算偏誤（中文字元 vs 英文詞密度不對等）
- **樣本限制**
  - basketball_influencer tags 全 NaN
  - food_review 99% 無 tag
  - restaurant_brand 僅 3 頻道（樣本量低）
- **方法限制**
  - LDA 在多語混雜語料上語言主導
  - 部分類別樣本小（如 Real Fake n=8），卡方檢定預期頻數可能 <5
  - 未做多重比較校正（10 RQ × 9 類別）
- **資料缺口**
  - 未取得 YouTube Studio 內部指標（impression、CTR、AVD）
  - 樣本時間點單一（2026-05），未控制季節性、節慶、政治事件
- **未來方向**：多語 embedding（XLM-R）、頻道隨機效應模型、時間序列、合作 YouTuber 取得內部指標

**視覺需求**
- 四象限版面：內容貼標限制 / 樣本限制 / 方法限制 / 未來方向
- 每象限 2–3 條 bullet，用 icon 強化視覺記憶

**講者備註**
- 老師最常問的問題就在這頁——主動講出來顯示成熟度
- 強調未來方向是「我們知道下一步該怎麼做」而非「我們做不到」

## P15 — 結論與建議

**講者要點**
- **五大核心結論**
  1. 類別溢價顯著（控制訂閱後仍有 ΔR² = +29.4%）
